Multi-column Layout

There are following multi-column properties:

  • column-count
  • column-gap
  • column-rule-style
  • column-rule-width
  • column-rule-color
  • column-rule
  • column-span
  • column-width

CSS Create Multiple Columns

The column-count property specifies the number of columns an element should be divided into.

CSS Column Rules

The column-rule-style property specifies the style of the rule between columns.

The column-rule-width property specifies the width of the rule between columns.

The column-rule-color property specifies the color of the rule between columns.

The column-rule property is a shorthand property for setting all the column-rule-* properties above.

The columns property is a shorthand property for column-width and column-count: columns: 100px 3;

Specify How Many Columns an Element Should Span

The column-span property specifies how many columns an element should span across. The following example specifies that the h2 element should span across all columns.

Specify The Column Width

The column-width property specifies a suggested, optimal width for the columns.

CSS column-fill Property

The column-fill property specifies how to fill columns, balanced or not.

Tip: If you add a height to a multi-column element, you can control how the content fills the columns.
