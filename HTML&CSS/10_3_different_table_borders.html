Table With Rounded Borders

Use the CSS border-radius property to add rounded corners to the borders.

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
John Doe 80

Skip the border around the table by leaving out border and border radius attribute from the css selector for "table" tag:

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
John Doe 80

With the border-color property, you can set the color of the border.

With the border-style property, you can set the appereance of the border. The following values are allowed:

Dotted

Dashed

Solid

Double

Groove

Ridge

Inset

Outset


Hidden

None

To set the width of a table, add the style attribute to the table element:
