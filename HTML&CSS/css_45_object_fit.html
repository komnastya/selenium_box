CSS The object-fit Property

The CSS object-fit property is used to specify how an <img> or <video> should be resized to fit its container.

The object-fit property can take one of the following values:

  • fill: the image is resized to fill the given dimension. If necessary, the image will be stretched or squished to fit (default).

    This picture has size of 400x379, but it is squished to 200x379:



  • contain: the image keeps its aspect ratio, but is resized to fit within the given dimension.

    This picture has size of 400x379, but it is resized to 200x379:



  • cover: the image keeps its aspect ratio and fills the given dimension. The image will be clipped to fit.

  • none: the image is not resized:
