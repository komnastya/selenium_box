Navigation Bar = List of links.

A navigation bar is basically a list of links, so using the <ul> and <li> elements makes perfect sense, the bullets should be removed from the list and the margins and padding should be changed.

display: block; displays the links as block elements makes the whole link area clickable (not just the text), and it allows us to specify the width (and padding, margin, height, etc. if you want)

Fixed Full-height Side Nav

Try to scroll this area, and see how the sidenav sticks to the page

Notice that this p element has a left margin of 25%. This is because the side navigation is set to 25% width. If you remove the margin, the sidenav will overlay/sit on top of this p.

Also notice that we have set overflow:auto to sidenav. This will add a scrollbar when the sidenav is too long (for example if it has over 50 links inside of it).
