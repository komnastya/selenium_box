CSS Text Shadow

The CSS text-shadow property applies shadow to text. In its simplest use, you only specify the horizontal shadow (2px) and the vertical shadow (2px), and add color:

This text shadow effect is 2px 2px orange

Then, add a blur effect to the shadow:

This text shadow effect is 2px 2px 5px orange

The following example shows a white text with black shadow:

This text shadow effect is 2px 2px 4px black

The following example shows a red neon glow shadow:

This text shadow effect is 0 0 4px red

Multiple Shadows

To add more than one shadow to the text, you can add a comma-separated list of shadows. The following example shows a red and blue neon glow shadow:

This text shadow effect is 0 0 3px #FF0000, 0 0 5px #0000FF;

You can also use the text-shadow property to create a plain border around some text (without shadows):

This text shadow effect is -1px 0 black, 0 1px black, 1px 0 black, 0 -1px black;
